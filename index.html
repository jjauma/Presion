Configuración inicial

Una sola vez. Después podrás añadir y ver tus lecturas sincronizadas en todos tus dispositivos.

Cómo crear tu proyecto de Firebase (paso a paso)
  1. Abre console.firebase.google.com e inicia sesión con tu cuenta de Google.
  2. Clic en "Añadir proyecto". Nombre (ej: mi-presion). Desactiva Analytics. Crear.
  3. Menú izquierdo → "Firestore Database" → "Crear base de datos". Región eur3. Modo de prueba. Habilitar.
  4. ⚙ → "Configuración del proyecto" → "Tus aplicaciones" → icono </>. Apodo. NO hosting. Registrar.
  5. Copia el objeto firebaseConfig = { ... } entero.
  6. Pégalo abajo.

Seguridad: el modo de prueba caduca en 30 días. Cambia las reglas a allow read, write: if true; antes.
